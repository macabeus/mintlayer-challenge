## Contribution

1 - Clone this repository

```
> git clone git@github.com:macabeus/mintlayer-challenge.git
> cd mintlayer-challenge
```

2 - Install its dependencies

```
> npm i
```

3 - Start the developer mode

```
> npm run start
```

4 - Build for production

```
> npm run build
```

### Lint

```
> npm run lint
```
